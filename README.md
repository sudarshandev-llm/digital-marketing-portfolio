# Team Alum — Digital Marketing Agency Portfolio

A modern, responsive portfolio website for **Team Alum**, a digital marketing agency specializing in social media growth and brand promotion.

## 🚀 Features

- **9 fully designed sections** — Hero, About, Services, Benefits, Requirement Form, Portfolio, Testimonials, FAQ, Contact
- **Dark mode support** — Toggleable theme persisted to localStorage
- **Animated counters** — Stats animate on scroll
- **Staggered reveal animations** — Cards fade in sequentially
- **Typing effect** — Hero tagline types itself on load
- **Floating particles** — Subtle background motion in hero
- **Gradient shifts** — Dynamic color transitions on stats banner
- **Fully responsive** — Mobile, tablet, and desktop layouts
- **SEO optimized** — Meta tags, Open Graph, semantic HTML
- **Marketing requirement form** — Detailed submission form with checkboxes for platforms, content types, and goals
- **Contact section** — Email, phone, and direct Gmail link

## 🛠 Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Flexbox, Grid, Keyframe animations
- **Vanilla JavaScript** — Intersection Observer, theme toggle, form handling, animated counters
- **Font Awesome 6** — Icons
- **Google Fonts (Inter)** — Typography
